## Installation

<p>1. Clone the repository</p>

```bash
git clone https://github.com/tiny-ideas-ir/lib-vue-components
```

<p>2. Create an `.npmrc` file in the root of the project and add the following
  line</p>

```bash
@tiny-ideas-ir:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_TOKEN
```

<p>
<span>3. Create a github token and replace `YOUR_GITHUB_TOKEN` with the token youcreated.</span>
<br/><span>Set the `read:packages` permission for the token.</span>
<br/><span>Watch this video: <a href="https://www.youtube.com/watch?v=iLrywUfs7yU" target="_blank">https://www.youtube.com/watch?v=iLrywUfs7yU</a></span>
</p>

<p>4.Install the package using npm or yarn</p>

```bash
yarn add @tiny-ideas-ir/lib-vue-components
```

<p>3. Import the package in your project.</p>

### vue3 

```javascript
import vueComponents from "@tiny-ideas-ir/lib-vue-components";

// install on vue app
vueApp.use(vueComponents);
```

### nuxt3
<p>Create a file called `component-library.client.ts` in plugin folder and add the following code</p>
```javascript
// plugins/component-library.client.ts
import { defineNuxtPlugin as init } from '@tiny-ideas-ir/lib-vue-components/nuxt';

export default defineNuxtPlugin((nuxtApp) => {
  return init(nuxtApp, { prefix: 'cl-' });
});
```
